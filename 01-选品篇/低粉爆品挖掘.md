# 低粉爆品挖掘

> 找到粉丝少但销量高的潜力爆品

---

## 核心逻辑

通过四维验证模型，挖掘0-1000粉账号的爆品机会。这类账号的特点是：
- 粉丝少（竞争小）
- 但内容已经验证过（点赞>10）
- 说明产品有市场，只是账号刚起步

---

## 四维验证模型

| 维度 | 指标 | 阈值 | 说明 |
|------|------|------|------|
| 内容维度 | 点赞数 | >10 | 内容有传播力 |
| 时间维度 | 发布时间 | 1-2个月内 | 需求正在爆发 |
| 供给维度 | 评论数 | <50 | 竞争度低 |
| 账号维度 | 粉丝数 | <1000 | 起步期账号 |

---

## 执行流程

### Step 1: 确定类目关键词

根据目标市场确定搜索类目，例如：
- 家居日用：厨房收纳、清洁工具
- 数码配件：充电宝、数据线
- 美妆护肤：化妆工具、护肤神器

### Step 2: 搜索爆款笔记

1. 打开目标平台（小红书/抖音等）
2. 搜索类目关键词
3. 按「最新」或「热度」排序
4. 筛选1-2个月内的帖子

### Step 3: 逐个验证账号

对于每个爆款帖子：
1. 点击作者头像进入主页
2. 查看粉丝数（必须<1000）
3. 查看点赞数（必须>10）
4. 确认发布时间（必须1-2个月内）

### Step 4: 记录数据

| 字段 | 说明 |
|------|------|
| 帖子ID | 唯一标识 |
| 标题 | 爆款标题 |
| 账号名称 | 作者 |
| 粉丝数 | 必须<1000 |
| 点赞数 | 必须>10 |
| 发布时间 | 1-2个月内 |
| 图片URL | 首图链接 |
| 视频URL | 视频链接（如有） |
| 帖子链接 | 原始链接 |

---

## 输出格式

```
Excel表格：低粉爆品数据_YYYYMMDD.xlsx

字段：帖子ID | 标题 | 账号 | 粉丝数 | 点赞数 | 发布时间 | 图片URL | 链接
```

---

## 注意事项

1. **粉丝数必须逐个验证** - 不能只看列表显示，要进主页确认
2. **时间放宽到2个月** - 如果需求持续存在
3. **多平台验证** - 同款产品在多平台出现说明需求真实
4. **持续跟踪** - 记录账号后续增长情况
