# 场景图制作

> 影视化拍摄逻辑，让产品图更有感染力

---

## 影视化标准

### 核心原则

```
场景图 = 产品(60%) + 使用场景(30%) + 情绪氛围(10%)
```

### 光线标准

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| 自然光 | 窗边、户外自然光 | 大部分产品 |
| 暖色调 | 2700K-3500K | 家居、厨房、美食 |
| 冷色调 | 5000K-6500K | 数码、科技产品 |

### 场景选择

根据产品类型选择场景：

| 产品类型 | 推荐场景 |
|----------|----------|
| 厨房用品 | 厨房台面、餐桌、户外烧烤 |
| 家居收纳 | 整洁的卧室、衣柜、书房 |
| 数码配件 | 简约桌面、咖啡厅、办公桌 |
| 美妆护肤 | 梳妆台、浴室、镜子前 |
| 母婴用品 | 婴儿房、客厅爬行垫 |
| 运动户外 | 健身房、公园、户外 |

### 情绪氛围

目标情绪：**快乐、自由、放松、品质感**

避免：
- 阴沉、压抑
- 过于生活化（脏乱差）
- 过度P图（失真）

---

## 构图公式

### 公式1: 黄金比例
- 产品放置在画面1/3或2/3处
- 留白30-40%给场景
- 视觉焦点明确

### 公式2: 俯拍平铺
- 45度俯拍
- 产品平铺展示
- 周边搭配相关道具

### 公式3: 人物带入
- 模特使用产品
- 表情自然放松
- 抓拍非摆拍

## AI场景图生成

### 提示词公式

```
[产品描述], placed on/in [场景描述], [光线描述], [情绪描述], high quality, photorealistic
```

### 示例

**产品**: 白色陶瓷咖啡杯
**场景图提示词**:
```
A white ceramic coffee cup on a minimalist wooden table, morning sunlight from window, cozy cafe atmosphere, warm tones, shallow depth of field, high quality, photorealistic
```

- [ ] 光线自然有氛围
