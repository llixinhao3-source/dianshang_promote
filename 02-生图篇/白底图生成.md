# 白底图生成

> 产品主图的标准规范

---

## 白底图标准

### 基础规格

| 参数 | 标准 |
|------|------|
| 背景色 | 纯白 (RGB: 255,255,255) |
| 产品占比 | 60-70% |
| 图片尺寸 | 1024×1024 或 1:1比例 |
| 文件格式 | PNG |
| 分辨率 | 无水印、无LOGO |

## 制作流程

### Step 1: 准备原图

- 确保产品主体清晰
- 光线均匀，无严重阴影
- 产品边缘可辨识

### Step 2: 背景移除

使用BiRefNet或在线工具移除背景

### Step 3: 边缘处理

- 羽化边缘（避免生硬切边）
- 补全透明区域
- 调整产品位置（居中或黄金比例）

### Step 4: 输出检查

- [ ] 背景是否完全白色
- [ ] 产品边缘是否自然
- [ ] 是否有残留背景色
- [ ] 产品是否完整

## 不同平台要求

### 亚马逊
- 主图：纯白背景，产品占85%
- 辅图：可带场景
- 尺寸：1500×1500以上

### 小红书
- 首图：尽量白底或简洁背景
- 辅图：场景图更佳
- 尺寸：3:4比例（竖版）

### 淘宝/天猫
- 主图：白底图必备
- 辅图：场景图+细节图
- 尺寸：800×800以上

## 常见问题

### Q: 产品有透明部分（玻璃杯、塑料盒）？
A: 背景移除后，透明部分保持透明PNG格式，或手动填充白色

### Q: 产品颜色接近白色？
A: 使用浅灰背景(RGB: 250,250,250)作为区分

### Q: 有复杂图案的产品？
A: 确保图案清晰可辨，必要时手动调整
