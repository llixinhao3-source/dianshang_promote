# 标题去AI化

> 把AI生成的标题转化成native speaker的自然表达

---

## AI痕迹特征

### 常见AI写法

| AI写法 | 问题 |
|--------|------|
| "The Best..." | 过度营销感 |
| "Top 10..." | 机械堆砌 |
| "Premium Quality..." | 夸大宣传 |
| "Introducing..." | 官方腔调 |
| 过度使用感叹号!!! | 视觉噪音 |
| 关键词堆砌 | 不自然 |

---

## 去AI化技巧

### 技巧1: 情绪词替换

| AI写法 | 去AI化 |
|--------|--------|
| "The Best Coffee Maker" | "The coffee maker I'm actually obsessed with" |
| "Top 5 Tips" | "5 things I wish I knew sooner" |
| "Premium Quality" | "feels way more expensive than it is" |

### 技巧2: 口语化表达

| AI写法 | 去AI化 |
|--------|--------|
| "It is very useful" | "for real, this thing is a game changer" |
| "This product is good" | "actually solid, no cap" |
| "You should buy this" | "just get it, trust" |

### 技巧3: 悬念感营造

| AI写法 | 去AI化 |
|--------|--------|
| "Features of X" | "I spent a month with X, here's what I actually think" |
| "How to use X" | "nobody talks about how wild X is" |
| "Benefits of X" | "X is fine, BUT..." |

## 爆款标题公式

### 公式1: 人群+痛点+方案

```
[目标人群]: [痛点描述], [你的产品] is all you need
```

示例：
```
For anyone who hates doing dishes - this drying rack is an absolute lifesaver
```

### 公式2: 场景+成果+情绪

```
Spent [时间] with [产品], [情绪爆发词]
```

示例：
```
Spent 2 weeks with this air fryer, my cooking has never hit different
```

### 公式3: 反差+好奇

```
I was wrong about [产品] until [触发事件]
```

示例：
```
I thought bidets were weird until I stayed at an Airbnb with one
```

### 公式4: 真实体验

```
[产品] after [时间段] - [真实反馈]
```

示例：
```
$30 face cream after 3 months - the before/after is actually insane
```

## 本地化要点

### 英式vs美式
- 美式: "Mom" / "Candy" / "Sneakers"
- 英式: "Mum" / "Sweets" / "Trainers"

### 俚语使用
- 适度使用当地俚语增加真实感
- 但不要过度，避免歧义

### 文化元素
- 参考目标市场的热点话题
- 融入当地节日、事件

---

## 检查清单

- [ ] 无"Best/Top/Premium"等AI词汇
- [ ] 语气自然，像真人写的
- [ ] 有情绪词但不过度
- [ ] 悬念感拉满
- [ ] 符合目标市场表达习惯
