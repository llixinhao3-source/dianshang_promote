# 亚马逊上架SOP

> 从选品到亚马逊Listing的完整流程

---

## 上架前准备

### 素材清单

- [ ] 白底图 × 6-7张（必须白底）
- [ ] 场景图 × 2-3张
- [ ] 尺寸图（带尺寸标注）
- [ ] 去AI化英文标题
- [ ] 五点描述（Bullet Points）
- [ ] 产品描述（A+内容）
- [ ] 关键词（Search Terms）

---

## 图片标准

### 亚马逊主图要求

| 参数 | 标准 |
|------|------|
| 背景 | 纯白 (RGB: 255,255,255) |
| 产品占比 | 85%以上 |
| 图片尺寸 | 最长边1500px以上 |
| 文件格式 | JPEG或TIFF |
| 分辨率 | 72dpi以上 |

### 图片顺序

1. **主图**：白底产品正面照
2. **图2**：产品侧面照
3. **图3**：产品背面照
4. **图4**：使用场景
5. **图5**：尺寸图或细节图
6. **图6**：包装内容物
7. **图7**：对比图或信息图

## 标题格式

### 标准公式

```
[品牌名] + [核心关键词] + [产品特性] + [规格/颜色] + [数量]
```

### 示例

```
AYSRTH Stainless Steel Coffee Mug with Lid, 12oz Double Wall Thermal Travel Tumbler, BPA-Free Insulated Cup for Office, Home and Outdoor (Black)
```

### 标题规则

- 长度：50-200字符（越短越好）
- 首字母大写
- 数字用阿拉伯数字（1 not one）
- 包含核心关键词
- 不含促销词（FREE/Best Seller）
- 不含特殊字符

## 五点描述（Bullet Points）

### 公式

```
功能1 → 解决痛点
功能2 → 具体参数
功能3 → 材质/工艺
功能4 → 使用场景
功能5 → 售后保障
```

### 示例

```
• 【Premium Quality】Made of 18/8 stainless steel, double-wall vacuum insulation, keeps drinks hot for 6 hours or cold for 12 hours

• 【 Leak-Proof Design】Secure lid with silicone seal, perfect for travel, commute, office and outdoor activities

• 【 BPA-Free & Eco-Friendly】Non-toxic food-grade materials, safe for hot and cold beverages

• 【 Perfect Gift Choice】Ideal for coffee lovers, outdoor enthusiasts, birthday, Christmas and holiday gifts

• 【 100% Satisfaction Guarantee】30-day money-back and 12-month warranty, customer satisfaction is our top priority
```

## 关键词策略

### 搜索词（Search Terms）

- 长度：≤250字节
- 用空格分隔
- 不重复标题中的词
- 包含同义词、长尾词
- 不含其他品牌名

### 例子

```
travel tumbler insulated cup coffee mug thermos flask water bottle sports bottle office mug
```

## 产品描述（A+ Content）

### 结构

```
【开头】产品定位（1段）
  ↓
【中间】核心卖点（3-4段）
  - 痛点引入
  - 产品优势
  - 使用场景
  ↓
【结尾】品牌介绍+行动号召
```

### 注意事项

- 使用HTML标签
- 避免过度营销
- 展示真实使用场景
- 包含尺寸对比

## 定价策略

### 成本计算

```
产品成本 + 国际运费 + 平台费(~15%) + 广告费 + 利润 = 售价
```

### 参考利润率

| 产品类型 | 建议利润率 |
|----------|------------|
| 低竞争蓝海 | 40%+ |
| 中等竞争 | 25-40% |
| 高竞争红海 | 15-25% |
