# 小红书发布SOP

> 完整的小红书笔记发布流程

---

## 发布前准备

### 素材准备清单

- [ ] 白底图 × 1-3张
- [ ] 场景图 × 3-5张
- [ ] 去AI化标题 × 1个
- [ ] 正文内容（痛点+解决方案）
- [ ] 热门话题标签 × 3-5个

### 账号检查

- [ ] 头像清晰
- [ ] 昵称易记
- [ ] 简介明确
- [ ] 已绑定店铺（如需）

---

## 标题创作

### 爆款公式

```
公式1: 人群+痛点 → [目标人群] + [痛点描述] + [产品名] is all you need

公式2: 场景+成果 → [使用场景] + [成果展示] + [情绪词]

公式3: 对比反差 → [我的预期] vs [实际结果] + [真实反馈]
```

### 标题示例

```
• 出租屋神器！这个收纳柜让我的小窝变大house
• 终于找到不踩雷的充电宝了！10000mAh能充3次
• 跟风买的吐司机，使用1个月真实反馈来了
```

---

## 正文写作

### 结构模板

```
【开头】痛点引入（1-2句）
  ↓
【中间】产品介绍（3-5句）
  - 外观/材质
  - 使用体验
  - 亮点功能
  ↓
【结尾】行动号召（1-2句）
  - 推荐指数
  - 适合人群
```

### 正文示例

```
之前一直觉得卧室东西太多找不到，
直到入了这个收纳柜！

✨ 尺寸刚好放在床头
✨ 奶油色系超好看
✨ 收纳空间比想象中大

用了半个月，真的香！
推荐给租房党和小卧室姐妹~
```

## 图片处理

### 首图要求
- 清晰明亮
- 产品突出
- 白底或简洁背景
- 可添加文字标签

### 图片顺序
1. 首图：产品全貌
2. 场景图：使用场景
3. 细节图：产品亮点
4. 对比图：使用前后（如有）
5. 截图：购买链接（如有）

## 话题标签

### 热门话题示例

| 类目 | 推荐话题 |
|------|----------|
| 家居 | #家居好物 #收纳神器 #租房改造 |
| 数码 | #充电宝推荐 #数码好物 #平价替代 |
| 美妆 | #护肤好物 #化妆工具 #平价护肤 |
| 母婴 | #母婴好物 #宝宝用品 #辅食工具 |

### 标签数量
- 必选：3-5个相关话题
- 建议：包含1个大热话题 + 2-3个精准话题

## 发布时间

### 黄金时段

| 时段 | 适合类目 |
|------|----------|
| 早7-9点 | 家居、美食、母婴 |
| 中12-14点 | 数码、穿搭 |
| 晚6-9点 | 全品类 |

### 推荐时间
- 工作日：晚6-8点
- 周末：早9-11点

---

## 发布后维护

### 24小时内
- 及时回复评论
- 引导互动（点赞/收藏）
- 更新高质量评论置顶

### 3天内
- 观察数据（浏览/点赞/收藏/评论）
- 分析数据不好的原因
- 决定是否重新发布

---

## 数据指标

| 指标 | 合格线 | 优秀线 |
|------|--------|--------|
| 点赞率 | >3% | >8% |
| 收藏率 | >5% | >10% |
| 评论率 | >0.5% | >1% |
