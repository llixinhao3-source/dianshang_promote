# 竞品监控

> 实时监控竞品价格、库存、评论变动

---

## 监控维度

| 维度 | 指标 | 告警条件 |
|------|------|----------|
| 价格监控 | 当前价格 | 变动>5% |
| 库存监控 | 库存数量 | 库存<20 |
| 评论监控 | 差评数/评分 | 差评增加/评分下降 |
| BSR监控 | 排名位置 | 排名变动>50名 |
| 促销监控 | 折扣/优惠券 | 发现限时促销 |

---

## 监控表格式

| 商品 | 原价 | 现价 | 库存 | 评分 | BSR | 差评数 | 监控时间 | 备注 |
|------|------|------|------|------|-----|--------|----------|------|
| | | | | | | | | |

---

## 执行流程

### 每日监控

```
1. 打开竞品链接
2. 记录当日数据
3. 对比昨日数据
4. 触发告警则通知
```

### 周度分析

```
1. 汇总本周数据
2. 分析竞品策略变化
3. 输出竞品周报
4. 调整自身运营策略
```

---
