# 客服质检SOP

> 自动检测聊天记录违禁词，降低违规风险

---

## 质检频率

| 场景 | 频率 | 说明 |
|------|------|------|
| 日常质检 | 每天23:00前 | 检测当天聊天 |
| 专项质检 | 不定期 | 根据投诉进行 |
| 新员工 | 上岗前3个月 | 100%质检 |

---

## 违禁词分类

### 严重违规 🔴

| 类型 | 违禁词示例 |
|------|------------|
| 辱骂攻击 | 傻逼、SB、滚、狗东西 |
| 人身攻击 | 不要脸、臭不要脸 |
| 敏感词 | 报警、微信、京东、抖音 |
| 推诿 | 我不管、随便你、不关我的事 |
| 威胁 | 投诉、举报、找平台 |
| 其他 | 骗子、骗人、诈骗 |

### 一般违规 ⚠️

| 类型 | 违禁词示例 |
|------|------------|
| 不耐烦 | 无语、服了、呵呵 |
| 催促 | 快点、赶紧、给钱 |
| 消极 | 随便、不买就走 |

---

## 质检维度

| 维度 | 权重 | 说明 |
|------|------|------|
| 服务态度 | 30% | 语气、措辞 |
| 响应速度 | 20% | 回复时长 |
| 问题解决 | 30% | 是否解决客户问题 |
| 规范用语 | 20% | 无违禁词 |

---

## 执行流程

```
1. 导出聊天记录（日期范围）
2. 读取聊天文本
3. 违禁词检测（正则匹配）
4. 生成违规报告
5. 推送给管理员
```

## 评级标准

| 评级 | 分数 | 说明 |
|------|------|------|
| ⭐⭐⭐⭐⭐ 优秀 | 90-100 | 无违规 |
| ⭐⭐⭐⭐ 良好 | 80-89 | 一般违规≤2次 |
| ⭐⭐⭐ 合格 | 70-79 | 一般违规≤5次 |
| ⭐⭐ 警告 | <70 | 有严重违规 |

## 处理措施

| 评级 | 处理措施 |
|------|----------|

| 良好 | 保持 |
| 合格 | 培训 |
| 警告 | 绩效扣罚+培训 |
