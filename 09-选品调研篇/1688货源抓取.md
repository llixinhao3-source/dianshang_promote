# 1688货源抓取

> 找到源头工厂，降低采购成本

---

## 抓取字段

| 字段 | 说明 |
|------|------|
| 商品标题 | 产品名称 |
| 价格 | 批发价 |
| 起订量 | MOQ |
| 供应商 | 厂家名称 |
| 所在地 | 工厂地址 |
| 成交量 | 反映供货能力 |
| 诚信等级 | 供应商可靠性 |

---

## 筛选标准

| 指标 | 建议值 | 说明 |
|------|--------|------|
| 诚信年限 | >3年 | 稳定供应商 |
| 成交率 | >80% | 高成交 |
| 复购率 | >20% | 产品认可 |
| 响应率 | >90% | 服务态度 |

---

## 工厂筛选

```
一手货源特征：
- 诚信年限>5年
- 工厂型供应商
- 支持定制
- 起订量灵活
- 价格有优势
```
