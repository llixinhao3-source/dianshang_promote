# 亚马逊选品

> BSR分析+利润计算，科学选品

---

## 选品维度

| 维度 | 指标 | 合格值 |
|------|------|--------|
| 销量 | 月销 | >300单 |
| 竞争度 | 评论数 | <100条 |
| 利润 | 毛利率 | >30% |
| 评分 | 平均分 | <4.3 |
| 趋势 | 排名变化 | 上升中 |

---

## BSR分析

```
Best Seller Rank (BSR)
- 前100名：红海，谨慎进入
- 100-500名：竞争激烈
- 500-2000名：有机会
- 2000名后：蓝海机会
```

---

## 利润计算

```
售价 - 采购成本 - 国际运费 - 平台费(~15%) - 广告费(~15%) = 利润

毛利率 = 利润 / 售价 × 100%
```
