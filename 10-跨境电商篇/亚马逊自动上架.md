# 亚马逊自动上架

> 批量创建Listing，提高效率

---

## 上架字段

| 字段 | 要求 | 说明 |
|------|------|------|
| 主图 | 白底 | 1500×1500 |
| 标题 | <200字符 | 核心词前置 |
| 五点 | 5条 | 特性+痛点 |
| 描述 | A+ | 场景化 |
| 关键词 | <250字节 | 长尾词 |

---

## 去AI化标题公式

```
[品牌] + [核心词] + [特性] + [规格] + [数量]

去AI化技巧：
- 避免：Best/Top/Premium
- 加入：actual/finally/no cap
- 口语化：trust/for real
```

---

## 批量上架流程

```
1. 整理产品数据（Excel）
2. 生成去AI化标题
3. 处理产品图片
4. 批量上传
5. 审核确认
```

---

## 质量检查

- [ ] 标题无AI痕迹
- [ ] 图片符合标准
- [ ] 关键词无重复
- [ ] 价格合理
- [ ] 库存充足
