# 速卖通抓取

> 竞品数据采集，分析市场

---

## 抓取字段

| 字段 | 说明 |
|------|------|
| 商品标题 | 英文标题 |
| 价格 | 美元售价 |
| 销量 | 月销量 |
| 评价数 | 评论总数 |
| 评分 | 平均分 |
| 商家 | 店铺名 |
| 上架时间 | 新品判断 |

---

## 筛选条件

| 指标 | 筛选值 | 说明 |
|------|--------|------|
| 销量 | >100/月 | 有市场 |
| 评论 | <100 | 竞争小 |
| 评分 | <4.3 | 痛点存在 |
| 上架 | <6个月 | 新品机会 |

---

## 用途

- 竞品分析
- 价格调研
- 选品参考
- 供应链对接
