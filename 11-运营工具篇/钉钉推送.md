# 钉钉推送

> 定时推送数据报告，自动通知

---

## Webhook配置

```python
import requests

webhook = "https://oapi.dingtalk.com/robot/send?access_token=你的token"

payload = {
    "msgtype": "text",
    "text": {"content": "消息内容"}
}

requests.post(webhook, json=payload)
```

---

## 消息类型

### 文本消息

```json
{
  "msgtype": "text",
  "text": {"content": "内容"}
}
```

### Markdown消息

```json
{
  "msgtype": "markdown",
  "markdown": {
    "title": "标题",
    "text": "# 标题\n内容"
  }
}
```

### 链接消息

```json
{
  "msgtype": "link",
  "link": {
    "title": "标题",
    "text": "描述",
    "messageUrl": "链接"
  }
}
```

---

## 推送场景

| 场景 | 时间 | 内容 |
|------|------|------|
| 早报 | 9:00 | 昨日数据汇总 |
| 巡店 | 每2小时 | 异常告警 |
| 日报 | 18:00 | 当日数据 |
| 周报 | 周一9:00 | 周度汇总 |

---

## 定时配置

```json
{
  "schedule": {
    "kind": "cron",
    "expr": "0 9 * * *",
    "tz": "Asia/Shanghai"
  }
}
```
