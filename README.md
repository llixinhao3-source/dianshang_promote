## 核心方法论

### 选品四维验证模型

| 维度 | 指标 | 阈值 |
|------|------|------|
| 内容维度 | 点赞数 | >10 |
| 时间维度 | 发布时间 | 1-2个月内 |
| 供给维度 | 评论数 | <50 |
| 账号维度 | 粉丝数 | <1000 |

### 选品评分模型

```
总分 = 销量分×0.3 + 利润分×0.25 + 竞争度×0.25 + 评分分×0.2

• 90+ 强烈推荐
• 80-89 推荐
• 70-79 一般
• <70 不推荐
```
