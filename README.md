## 核心方法论

### 选品四维验证模型

| 维度 | 指标 | 说明 |
|------|------|------|
| 内容维度 | 点赞>10 | 内容有传播力 |
| 需求维度 | 1-2个月内 | 需求正在爆发 |
| 供给维度 | 评论<50 | 竞争度低 |
| 视觉维度 | 首图清晰 | 产品卖点突出 |

### 选品评分模型

```
总分 = 销量分×0.3 + 利润分×0.25 + 竞争度分×0.25 + 评分分×0.2

• 90+ 强烈推荐
• 80-89 推荐
• 70-79 一般
• <70 不推荐
```
